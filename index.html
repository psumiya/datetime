Business Date Library

This library wraps the Java 8 Date Time API and provides business methods used frequently in the enterprise applications world.

The library is evolving. Here are some methods this library provides currently:

  • Current Date
  • Yesterday's Date
  • Tomorrow's Date
  • Day after Tomorrow's Date
  • First Date of Current Month
  • Last Date of Current Month
  • First Date of Next Month
  • Last Date of Next Month
  • Find Nth Weekday in Given Month of Given Year
